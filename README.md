# 🌸 Цветочный магазин на Vue 3

Нежный интернет-магазин цветов, созданный с любовью и розовым настроением.

---

## 🌷 О проекте

Этот проект — учебный интернет-магазин цветов, разработанный на Vue 3 с использованием Vuetify 3. Здесь можно просматривать букеты, добавлять новые и оформлять заказы.

---

## ✨ Что внутри

| Раздел | Описание |
|--------|----------|
| **Главная страница** | Карусель с хитами продаж и сетка всех букетов |
| **Каталог букетов** | Карточки с фото, названием, описанием и ценой |
| **Добавление букета** | Форма с валидацией для новых букетов |
| **Мои букеты** | Список всех созданных букетов |
| **Детальная страница** | Подробная информация о каждом букете |
| **Дизайн** | Нежно-розовая тема во всём приложении 🌺 |

---

## 🛠 Технологии

- **Vue 3** — основной фреймворк
- **Vuetify 3** — библиотека компонентов
- **Vue Router** — навигация между страницами
- **GitHub** — хранение кода

---

## 🚀 Как запустить проект

```bash
# Скопировать репозиторий
git clone https://github.com/sssabbe/aip_sem4_flower_shop_sabi_abi_vue3.git

# Перейти в папку
cd flower-shop

# Установить зависимости
npm install

# Запустить проект
npm run serve
